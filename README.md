# Quiz Application (Java)

## Overview

This project is a **Java-based Quiz Application** that allows users to log in, read quiz rules, answer questions, and view their final score.
The application is designed to test the user's knowledge through multiple-choice questions.

## Project Files

The project contains the following Java files:

* **Login.java** – Handles user login and starts the quiz process.
* **Rules.java** – Displays quiz rules and instructions for the user.
* **Quiz.java** – Contains the quiz questions and manages the quiz logic.
* **Score.java** – Calculates and displays the final score after the quiz is completed.

## Features

* User login system
* Display quiz rules before starting
* Multiple-choice questions
* Automatic score calculation
* Final result display

## Technologies Used

* Java
* Java Swing / Java AWT (for GUI) or Console-based Java
* Object-Oriented Programming (OOP)

## How to Run the Project

1. Make sure **Java JDK** is installed on your system.
2. Place all `.java` files in the same folder.
3. Open terminal or command prompt in the project folder.
4. Compile the files:

```
javac Login.java Rules.java Quiz.java Score.java
```

5. Run the program:

```
java Login
```

## Project Flow

1. User starts the application using **Login.java**.
2. The **Rules page** explains the instructions of the quiz.
3. The **Quiz page** displays questions and collects answers.
4. The **Score page** shows the final result.
